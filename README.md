# Expense Tracker 

## Introduction

Expense tracker developed using JavaScript, HTML and CSS. You can add, edit and delete expenses. Also  create, list, delete categories and then tag them to each item. You can also generate  an expense report which gives you  a bar chart unsign ChartjsÂ´s library. It presents an accrual of all expenses per category in the specified date range.

# Configuration

Follow these steps to execute this project:

- Clone the repository
- Execute the HTML file with your browser
